About This Site

Welcome to "The Real Cory Sanders" webpage. I built this site from scratch specifically as a hub to connect anything and everything I create or collaborate on. Although not perfect, I wanted to put out my own site so people could connect with me and maybe share their journey as well. I will be adding to, enhancing, and upgrading my site as I learn more but for now, this is my best foot forward. Below are some links to some projects I've worked on plus some of the websites I use for reference. These resources are beyond helpful so feel free to check these out by clicking on the icon!

Music

I love to create music! I listen to almost everything, but I love the Rock and Metal genres the most. Here are a few songs I have on my Soundcloud. Give a thumbs up if you like it!

Projects

If you haven't used Codepen yet, I highly recommend it! It's a free "wysiwyg" (what you see is what you get) code editor for html, css, and JS. I like to post projects I've been working on here as well.

Github

I'm still learning the proper way to use Github but I have found it extremely useful for storing projects so I can access them later from anywhere I may be.

W3 Schools

I probably use this site more than any other for reference. It is a vast library of HTML, CSS, and JS tutorials. They offer lessons and even have website templates you can use to build a site of your own.

CSS Tricks

CSS Tricks is another helpful site that gives full, easy to understand tutorials on various CSS topics. This one should also be in your bookmarks.

MDN Web Docs

Like W3 Schools, the Mozilla Developer Network is an immense library should you require any reference. MDN Web Docs covers a wider variety of topics, however ranging from F.E.D. to game development.

Codecademy

A great learning site if you are wanting to take some free classes. The lessons are just challenging enough with an emphasis on looking for the answers yourself, which you will be doing a lot of as a coder.

freeCodeCamp

The material is presented in a slightly different format but the lessons don't require as much research to solve, like in Codecademy, while still maintaining a challenge.

Treehouse

Treehouse is a paid service but I find the lessons more to my speed of learning as well as affordable. The lessons are setup like a classroom with an instructor, teacher's notes, quizzes, and practice lessons to reenforce what you just learned. They also provide a built-in text editor so you can follow right along!
